# 🌐 Anusha Reddy - Portfolio Website

This is a personal **Portfolio Website** built with **HTML5 + CSS3**.  
It showcases **About Me, Education, Skills, Projects, Contact Form, and Social Links** with a **responsive design**.

---

## 🚀 Features

- Responsive Navbar with Resume Button  
- Hero Section with intro  
- About Section with professional description  
- Education Timeline (B.Tech, Intermediate, Schooling)  
- Skills Section (Technical + Soft Skills)  
- Projects Section (Daran Music, Task Manager, Responsive Blog)  
- Contact Form with input validation  
- Social Media Icons (LinkedIn, Email, GitHub) with hover effects  
- Hover, Focus, and Active states for accessibility  
- Fully responsive layout using **Flexbox + CSS Grid + Media Queries**

---

## 🛠️ Tech Stack

- **HTML5**  
- **CSS3 (Flexbox, Grid, Media Queries)**  
- **Font Awesome (for icons)**
